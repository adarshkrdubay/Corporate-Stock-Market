# 📈 Corporate-Stock-Market

A high-performance algorithmic stock market simulator built for competitive trading, tech events, and learning environments.  
Trade synthetic Indian equities, react to real-time news events, and compete to become the top broker.

---

## 🚀 Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Flask  
- **Database:** SQLite3  
- **Hosting:** PythonAnywhere  

---

## 🎯 Features

- ⚡ Algorithm-based stock price simulation  
- 📰 Real-time news-driven market events  
- 🏆 Competitive leaderboard system  
- 📊 Portfolio tracking and trading system  

---

## 💡 Use Cases

- Tech fests & competitions  
- Educational simulations  
- Trading strategy practice  
- Fun competitive environments  

---

## ⚠️ Disclaimer

This project is built for **educational and simulation purposes only**.  
No real money or real stock trading is involved.
